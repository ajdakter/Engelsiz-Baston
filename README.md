# Engelsiz-Baston
*Engelsiz Baston* is a mobile(android) application for blind people.
 Goals of *Engelsiz Baston* :
 ```
- Easily walk
- Listen book and news
- Share emotion about application(feedback)
- Easily learn features about application

```
